# Drag & Drop Image App

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Open http://localhost:3000 in your browser

## Features

- 80/20 split layout
- 4 grid containers in the canvas area (80%)
- Draggable images in the options panel (20%)
- Drag images from options panel and drop into any container
- Images automatically fit the container (object-fit: cover)
- Clear button to remove images from containers
- Replace existing images by dropping new ones

## How to Use

1. Drag any image from the right panel (Options - 20%)
2. Drop it into any of the 4 containers in the left area (Canvas - 80%)
3. The image will automatically fit the container
4. Click the Ã— button to remove an image
5. Drop a new image to replace an existing one
